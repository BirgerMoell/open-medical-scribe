This document is the starter quality management system index for the OMS Class I Web Reference Build.

It is not yet a complete QMS. It is the controlled-document map for the procedures, records, and technical-file elements needed for the initial MDR Class I path.

## Scope

Applies to the manufacturer-controlled OMS Class I Web Reference Build and the quality activities of the legal manufacturer for that regulated configuration.
