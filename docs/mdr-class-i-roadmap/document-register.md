# OMS Class I Document Register

Document ID: OMS-MDR-CL1-DOCUMENT-REGISTER
Version: 0.1
Status: Draft
Owner: Birger Moëll

Last updated: 2026-03-11

This register lists the first document set needed for an ordinary Class I MDR path. Status values below are placeholders and should be updated as documents are created.

| Document | Purpose | Initial status |
| --- | --- | --- |

| Device description and product specification | Define the reference build, variants, and architecture | Draft started |
| QMS index | List controlled procedures and records | Draft started |
| PRRC appointment record | Show Article 15 coverage | Draft started |
| Risk management plan | Define risk process and acceptability criteria | Draft started |
| Hazard analysis / risk file | Document hazards, controls, and residual risks | Draft started |
| Annex I GSPR checklist | Show conformity to general safety and performance requirements | Draft started |
| Software requirements specification | Define product behavior and constraints | Draft started |
| Software architecture description | Define modules, interfaces, and deployment | Draft started |
